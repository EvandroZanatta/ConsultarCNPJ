Consultar CNPJ

Acesse informações detalhadas sobre empresas brasileiras de forma rápida e confiável. Obtenha dados como nome da empresa, endereço, status do registro e mais, tudo por meio desta API acessível e fácil de usar.

O banco de dados possui 54.982.272 registros de empresas no Brasil.

Como Usar a API

Aprenda como consultar CNPJs por meio de nossa API.

Autenticação

Nenhuma autenticação é necessária para usar a API.

Limites

Não há limites no uso da API. Para manter a qualidade do serviço, pedimos gentilmente que você não faça muitas solicitações simultâneas à API. É aconselhável enviar solicitações uma após a outra, especialmente ao consultar informações para muitos CNPJs.

Nossa API é totalmente gratuita para uso.

Pontos de Acesso

A API pode ser acessada na seguinte URL:
GET https://opencnpj.inapplet.com/api/cnpj/

A resposta será no formato JSON.

GET /cnpj/:cnpj

A solicitação deve usar o método GET e fornecer o CNPJ de 14 dígitos, preferencialmente sem caracteres especiais.

Veja um exemplo de recuperação de dados clicando no link: demo /api/cnpj/.

Para detalhes sobre campos e valores, consulte a documentação oficial da "Receita Federal".
